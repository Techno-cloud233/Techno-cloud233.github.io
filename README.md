# 黑湖科技 前端样式示例

简单静态前端演示，参考现有官网视觉与内容（中文）。

快速预览

1. 在项目根目录打开：

```bash
open "blacklake-frontend/index.html"
```

或使用本地静态服务器：

```bash
# Python 3
python3 -m http.server 8080 --directory "blacklake-frontend"
# 然后访问 http://localhost:8080
```

结构

- `index.html` — 页面主文件
- `css/styles.css` — 样式
- `js/main.js` — 简单交互（移动菜单、平滑滚动）

说明

这是一个轻量示例，用于快速展示与原站点相似的 Hero、要点卡片与在线咨询浮动入口。可根据需要替换插画、文案与对接真实表单/后端接口。
